# ip-storyboard-creator

> 一键IP二创剧本+分镜表 — 从原始IP到飞书多维表格分镜表的全流程自动化技能

## 它能做什么

输入一个IP（电视剧/小说/角色），经过7个标准化步骤，输出一份可直接用于拍摄的飞书多维表格分镜表：

1. **原始IP输入** — 提供名称/梗概/剧本
2. **二创方向策划** — AI穷举5-8个方向供选择
3. **方向确认** — 人类决策
4. **写二创剧本** — 电影级标准（角色小传/潜台词对白/配乐建议/转折点/结尾余味）
5. **剧本确认** — 人类审核修改
6. **拆分分镜表** — 15秒Batch规则 + 运镜差异化设计
7. **写入飞书多维表格** — 21列结构化数据自动写入

## 核心优势

- **人机协作**：在需要审美的地方让人决策，不需要的地方AI全速推进
- **可拍摄标准**：剧本不是故事概要，是可直接拍摄的详细脚本
- **21列标准化字段**：从集数到关键帧提示词，一步到位
- **飞书原生集成**：最终输出在飞书多维表格，随时编辑分享
- **实战验证**：已通过《大奉打更人》《芈月传》两轮全流程验证

## 使用方式

### 前置条件

- OpenClaw 运行环境
- 飞书账号（用于多维表格写入）
- lark-cli 已配置（用于以用户身份创建和写入多维表格）

### 目录结构

```
ip-storyboard-creator/
  SKILL.md     — 技能主文件（7步全流程规范）
  README.md    — 本文档
  evals/
    evals.json — 评测数据
```

### 安装

将 `ip-storyboard-creator/` 目录放入 OpenClaw 的 `skills/` 目录下，重启 Gateway 即可。
